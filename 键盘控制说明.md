# 键盘控制说明 / Keyboard Control Guide

> 适用于无手柄环境下的 MuJoCo 仿真部署

## 启动方式

### 方式 1: 使用启动脚本（推荐）
```bash
cd /home/abc/RoboMimic_Deploy
./start_keyboard.sh
```

### 方式 2: 手动启动
```bash
cd /home/abc/RoboMimic_Deploy
conda activate robomimic
USE_KEYBOARD=1 python deploy_mujoco/deploy_mujoco.py
```

**重要提示**: 启动后会打开两个窗口，必须点击 **Pygame 小窗口（内含虚拟键盘实时高亮）** 才能输入！

## 键盘按键映射表

### 功能按键

| 手柄按钮 | 键盘按键 | 按钮 ID | 功能说明 |
|---------|---------|---------|---------|
| **START** | `Enter` (回车键) | 7 | 进入位控模式 (Position Control) |
| **SELECT** | `Backspace` | 6 | 退出到阻尼保护模式 (Passive Mode) |
| **A** | `Space` (空格键) | 0 | 配合 R1 进入行走模式 (LocoMode) |
| **B** | `B` | 1 | 配合 R1 触发踢腿动作 (Kick) |
| **X** | `X` | 2 | 配合 R1 进入舞蹈模式 (Dance) |
| **Y** | `Y` | 3 | 配合 R1/L1 触发武术动作 (KungFu) |
| **L1 (LB)** | `Tab` | 4 | 左肩键 (组合键) |
| **R1 (RB)** | `Left Shift` | 5 | 右肩键 (组合键) |
| **L3** | `C` | 8 | 左摇杆按下 |
| **R3** | `V` | 9 | 右摇杆按下 |

### 摇杆控制 (轴)

| 摇杆 | 键盘按键 | 轴 ID | 功能说明 |
|------|---------|-------|---------|
| **左摇杆 X 轴** | `A` / `D` 或 `←` / `→` | 0 | 左右移动 |
| **左摇杆 Y 轴** | `W` / `S` 或 `↑` / `↓` | 1 | 前后移动 |
| **右摇杆 X 轴** | `J` / `L` | 2 | 左右转向 |
| **右摇杆 Y 轴** | `I` / `K` | 3 | 视角/姿态调整 |
| **LT (左扳机)** | `Q` / `E` | 4 | 辅助控制 |
| **RT (右扳机)** | `U` / `O` | 5 | 辅助控制 |

## 标准操作流程

### 1. 启动与站立
```
1. 运行脚本后，会打开两个窗口：
   - MuJoCo 仿真窗口 (大窗口)
   - Pygame 键盘输入窗口 (小窗口)
   
2. 用鼠标点击 Pygame 小窗口，使其获得焦点（边框会高亮）

3. 按 Enter (START) → 终端显示 ">>> 切换到: POS_RESET 位控模式"

4. 按住 Left Shift (R1) + Space (A) → 终端显示 ">>> 切换到: LOCO 行走模式"

5. 按 Backspace (SELECT) → 机器人站立（如在仿真窗口按此键无效，请先点击小窗口）

6. 使用 WASD/方向键 控制行走，J/L 控制转向
   - 如果 WASD 无效，说明小窗口失去焦点，重新点击它
```

### 2. 查尔斯顿舞蹈 (推荐)
```
前提: 在 LocoMode 且机器人已站立
操作: 按住 Left Shift (R1) + X → 开始跳舞
退出: 
  - 按 Backspace (SELECT) → 阻尼保护模式
  - 舞蹈结束后按住 Left Shift (R1) + Space (A) → 恢复行走
```

### 3. 武术动作 (仅限仿真)
```
前提: 在 LocoMode 且机器人已站立
操作:
  - 按住 Left Shift (R1) + Y → KungFu 动作
  - 按住 Tab (L1) + Y → KungFu2 动作 (训练失败版)
```

### 4. 踢腿动作 (仅限仿真)
```
前提: 在 LocoMode 且机器人已站立
操作: 按住 Left Shift (R1) + B → Kick 动作
```

### 5. 紧急停止
```
按 Backspace (SELECT) → 立即切换到阻尼保护模式
或关闭 MuJoCo 窗口
```

## 常见问题

### ⚠️ Q: 按键没反应？（最常见问题）
**A**: **必须先点击 Pygame 小窗口使其获得键盘焦点！**

程序会打开**两个窗口**：
1. **MuJoCo 仿真窗口** (大窗口，显示机器人)
2. **Pygame 键盘输入窗口** (小窗口 240x120，标题 "Keyboard Joystick")

**正确操作流程：**
```
1. 启动程序后，两个窗口都会打开
2. 用鼠标点击 Pygame 小窗口（标题栏或窗口内）
3. 确保小窗口边框高亮（表示获得焦点）
4. 现在按键盘，终端会显示 ">>> 切换到: XXX 模式"
5. 如果按键后没反应，重新点击小窗口
```

### Q: 如何查看实时按键状态？
**A**: 在 `deploy_mujoco/deploy_mujoco.py` 中添加调试输出:
```python
# 在主循环中添加
if joystick.is_button_pressed(7):
    print("START button pressed!")
```

### Q: 想用真手柄怎么办？
**A**: 不加 `USE_KEYBOARD=1` 环境变量，脚本会自动检测手柄:
```bash
conda activate robomimic
python deploy_mujoco/deploy_mujoco.py  # 自动使用手柄 (如果有)
```

### Q: 如何在无显示器环境运行？
**A**: 设置虚拟显示驱动 (但键盘控制不可用):
```bash
export SDL_VIDEODRIVER=dummy
python deploy_mujoco/deploy_mujoco.py
```
